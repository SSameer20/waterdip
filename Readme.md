# Waterdip LABS Assignment

This project involves building a React.js with TypeScript dashboard for visualizing hotel booking data. The goal is to create an interactive single-page dashboard that allows users to explore booking data with dynamic filtering based on date ranges. The dashboard will contain four types of charts, which will automatically update based on the selected date range.

## Deployment

To deploy this project run

```bash
  git clone https://github.com/SSameer20/waterdip.git
```

```bash
  npm install
```

```bash
  npm run dev
```

or visit Website Directly
